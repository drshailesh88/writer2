# V1 Drafts

![CI Pipeline](https://github.com/drshailesh88/writer2/actions/workflows/ci.yml/badge.svg)

AI-powered research paper writing platform for medical students.

## Development

```bash
npm install
npm run dev
```

## Testing

```bash
npm run lint          # ESLint
npx tsc --noEmit      # Type check
npm run test          # Vitest unit tests
npm run test:e2e      # Playwright E2E tests
```
